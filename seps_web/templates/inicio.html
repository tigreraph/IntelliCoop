IntelliCoop

Sistema Inteligente de Análisis y Riesgo Cooperativo para las cooperativas de ahorro y crédito supervisadas por la SEPS Ecuador, con metodología CAMEL, inteligencia artificial y modelos predictivos.

Ver Dashboard Chat Rakkun IA

¿Qué es IntelliCoop?

IntelliCoop es una plataforma web de análisis financiero que procesa los Estados Financieros Mensuales (EEFF) de las cooperativas de ahorro y crédito ecuatorianas reportadas a la SEPS, aplicando la metodología CAMEL para evaluar la salud financiera de cada entidad.

El sistema integra un asistente de inteligencia artificial (Rakkun) que permite consultar la base de datos mediante lenguaje natural, transformando preguntas en SQL automáticamente usando el modelo DeepSeek V3 via OpenRouter.
